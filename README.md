# InstagramCloneSwiftUI

This repository was created following the tutorial that you can find in [AppStuff](https://www.youtube.com/watch?v=7UKUCZuaVlA) YouTube channel. It was rewritten by me, refactoring some files, avoiding boilerplate code, following best practices and adding new features that in the tutorial you are not going to find. If you want to see how it was written you can visit the [Pull Requests](https://github.com/AlejandroRoblesA/InstagramCloneSwiftUI/pulls?q=is%3Apr+is%3Aclosed) created for this repository.
